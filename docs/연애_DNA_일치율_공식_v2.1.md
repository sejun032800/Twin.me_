# 연애 DNA 일치율(Romantic DNA Compatibility Score, RDCS) v2.1
## — 적응형 인터뷰 프로토콜과 애니어그램 날개(Wing) 결합확률을 포함한 통합 명세서

**분야:** 계산심리학(Computational Psychology) / 적응형 검사이론(Computerized Adaptive Testing) / 응용 통계
**버전:** v2.1 (v2.0을 대체하는 통합 문서)
**개정 사유:** v2는 "10분 인터뷰"를 전제로 궁합 점수 산출 로직만 정의했다. 실제 온보딩에서는 (1) 인터뷰 시간이 **최대 5분**으로 단축되어야 하고, (2) 애니어그램은 코어유형뿐 아니라 **날개(wing)**까지 다봉적(multimodal)으로 추론되어야 하며, (3) 자기보고 MBTI를 인터뷰 데이터로 검증하는 보조 출력이 필요하다. v2.1은 이 세 요구사항을 v2의 궁합 수식 체계와 하나의 파이프라인으로 통합한다.

## Abstract

본 문서는 두 사람 각각에 대해 (i) MBTI 자기보고로부터 결정론적 초기 사전분포를 구성하고, (ii) **최대 5분, 엔트로피 기반 그리디 알고리즘으로 질문 순서를 실시간 결정하는 적응형 인터뷰**를 통해 이를 베이지안 갱신하며, (iii) 애니어그램을 **코어유형(9)×날개(2) 결합확률(18칸)**로 정밀화하고, (iv) 부산물로 MBTI 축을 역추정하는 전체 개인 분석 파이프라인을 정의한다. 이렇게 산출된 두 사람의 사후 벡터는 v2의 궁합 결합식(Big Five 유사도-수준 분리, 애니어그램 코어+날개, 스턴버그 성향-상태 블렌드, 애착 정식 가중항)에 그대로 대입되어 최종 `DNA%`를 생성한다. 인터뷰 알고리즘의 이론적 근거는 적응형 검사이론(CAT)의 최대정보 문항선택 기준(maximum information item selection)에 있다.

# Part I — 개관

## 1. 표기법 및 전체 파이프라인

| 기호 | 정의 |
|---|---|
| `b = (O,C,E,A,N) ∈ [0,1]⁵` | Big Five 벡터 |
| `p ∈ Δ⁹` | 애니어그램 코어유형 사후확률 |
| `q ∈ Δ¹⁸` | 애니어그램 코어×날개 결합확률 *(신규)* |
| `r ∈ Δ⁹` | 날개 "번호"에 대한 주변확률(marginal) *(신규)* |
| `ψ = (anx, avo)` | 애착 불안·회피 연속벡터 |
| `U_d ∈ [0,1]` | 차원 `d`의 정규화 불확실성 지표 *(신규)* |

| `DNA%` | 최종 지표 ∈ [50,100] |

## 2. MBTI 기반 초기 사전분포

*(v2 Part I §2-4와 동일한 메커니즘 — 완전한 유도는 v2 문서 참조. 본 문서에서는 이후 §5의 불확실성 계산에 필요한 만큼만 압축해 재수록한다.)*

**Big Five 초기값:**
```
O = 0.5 + 0.15·sign(N/S; N=+1,S=−1)     C = 0.5 + 0.15·sign(J/P; J=+1,P=−1)
E = 0.5 + 0.15·sign(E/I; E=+1,I=−1)     A = 0.5 + 0.15·sign(T/F; F=+1,T=−1)
N = 0.5 + 0.05·sign(T/F;F=+1,T=−1) + 0.05·sign(E/I;I=+1,E=−1)
```

**애착 초기값** *(v2 §4)*: `anx⁽⁰⁾ = N⁽⁰⁾`,  `avo⁽⁰⁾ = 1 − (E⁽⁰⁾+A⁽⁰⁾)/2`

**애니어그램 원형 앵커행렬 E_i (9×5, Big5 공간)** — 애니어그램-FFM 상관연구(Wagner, 1981; Newgent et al., 2004)에 기반한 9유형별 대표 좌표:

| 유형 | O | C | E | A | N |
|---|---|---|---|---|---|
| 1 (개혁가) | 0.45 | 0.80 | 0.45 | 0.45 | 0.55 |
| 2 (조력가) | 0.50 | 0.40 | 0.65 | 0.80 | 0.55 |
| 3 (성취가) | 0.50 | 0.75 | 0.75 | 0.45 | 0.40 |
| 4 (개인주의자) | 0.75 | 0.35 | 0.40 | 0.50 | 0.70 |
| 5 (탐구자) | 0.75 | 0.55 | 0.25 | 0.40 | 0.55 |
| 6 (충성가) | 0.50 | 0.65 | 0.45 | 0.60 | 0.70 |
| 7 (열정가) | 0.75 | 0.35 | 0.80 | 0.55 | 0.35 |
| 8 (도전자) | 0.45 | 0.60 | 0.75 | 0.30 | 0.35 |
| 9 (평화주의자) | 0.50 | 0.40 | 0.35 | 0.75 | 0.35 |

**애니어그램 코어 초기값**: 위 앵커까지의 softmax 거리로 계산한다:
```
p_i⁽⁰⁾ = exp(−‖b⁽⁰⁾−E_i‖²/2τ_c²) / Σⱼ(...),   τ_c = 0.35
```

# Part II — 적응형 인터뷰 프로토콜

## 3. 이론적 근거: 적응형 검사이론과 최대정보 문항선택

고정된 순서로 모든 문항을 묻는 대신, **컴퓨터화 적응형 검사(Computerized Adaptive Testing, CAT)** 이론의 핵심 원칙 — 매 시점 가장 많은 정보를 줄 것으로 기대되는 문항을 선택한다 — 을 채택한다(Weiss, 1982; van der Linden & Glas, 2010). 표준 CAT은 문항반응이론(IRT)의 문항정보함수를 쓰지만, 본 시스템은 이미 정규-정규 켤레 및 심플렉스 사후분포 구조를 갖추고 있으므로 **문항정보 대신 사전분포의 불확실성 자체를 문항선택 기준으로 대체**한다 — 이는 "현재 가장 모르는 것을 먼저 묻는다"는 CAT의 직관을 그대로 유지하는 단순화다.

이 설계의 실질적 효과는 두 가지다:
- MBTI가 강하게 시사하는 차원(O,C,E,A의 주축)은 인터뷰 자원을 아낀다.
- MBTI가 침묵하는 차원(신경성, 애착, 애니어그램)에 시간을 집중한다.

## 4. 시간 예산 모델 (5분 상한)

한국어 자연 대화 속도 기준, 인트로/마무리 제외 실질 Q&A 예산은 약 280초다. 문항을 두 유형으로 나눈다 (날개 관련 문구는 §7에서 명시하듯 인터뷰 턴이 아니라 결과 화면 확인용 UX이므로 시간예산에서 제외):

| 유형 | 대상 | 턴당 시간 | 특징 |
|---|---|---|---|
| A. 서사형 개방질문 | Big5, 애착, 애니어그램 코어 | ~30초 | 엔트로피 기준 선택 대상 |
| C. 관계상태 직답 | 스턴버그 I/P/C | ~10초 | 엔트로피와 무관, **항상 고정 포함** (성향이 아닌 "지금 이 관계"를 묻는 질문이라 사전분포로 대체 불가) |

**기본 배분:** 서사형 6턴(180초) + 관계상태 3턴(30초) = 210초 사용, 70초는 여유(추가 확인질문 또는 자연스러운 대화 템포).

## 5. 정규화 불확실성 지표와 그리디 선택 알고리즘

### 5.1 연속차원(Big5, 애착)의 불확실성

정규-정규 켤레 사전분포의 폭 `σ_prior(d)`를, MBTI가 해당 차원에 얼마나 직접적으로 대응하는지에 따라 **차등 설정**한다 (v2는 전 차원에 균일한 σ=0.15를 썼으나, 이는 엔트로피 기반 선택이 무의미해지는 문제가 있어 v2.1에서 차등화한다):

| 차원 | σ_prior | 근거 |
|---|---|---|
| O, C, E, A | 0.12 | MBTI 단일 축에 직접 대응 (±0.15 오프셋) |
| N | 0.18 | 두 축의 약한 2차보정만 존재 |
| anx | 0.20 | N으로부터의 간접 유도 (r≈0.5) |
| avo | 0.22 | E,A 평균으로부터의 가장 약한 간접 유도 (r≈0.2~0.3) |

정규분포의 미분 엔트로피 `H = 0.5·ln(2πe·σ²)`는 σ에 대해 단조증가하므로, 순위 목적에는 분산 자체를 정규화해 사용하면 충분하다:

```
U_d = σ_prior(d)² / σ_max²,     σ_max = max(σ_prior) = 0.22        ... (1)
```

### 5.2 애니어그램 코어의 불확실성

```
U_EN = H(p⁽⁰⁾) / ln(9),     H(p) = −Σᵢ pᵢ ln pᵢ                    ... (2)
```

(섀넌 엔트로피를 이론적 최대치 `ln9`로 정규화 — 균등분포일 때 `U_EN=1`.)

### 5.3 그리디 선택

```
FOR turn = 1 to (예산 소진 or 조기종료):
    target ← argmax_d U_d   (아직 서사형 질문을 안 받은 차원 중)
    질문(target) 제시 → 답변 수집 → §8 갱신 → U_target 재계산
```

### 5.4 예시 초기 순위 (ENFP 자기보고 사용자 기준, 실계산값)

| 순위 | 차원 | U_d |
|---|---|---|
| 1 | 회피(avo) | 1.000 |
| 2 | 애니어그램 코어 | 0.969 |
| 3 | 불안(anx) | 0.826 |
| 4 | 신경성(N) | 0.669 |
| 5~8 | O, C, E, A (동률) | 0.298 |

→ 5턴의 서사형 질문 예산이 있다면, 이 사용자는 **회피 → 애니어그램 코어 → 불안 → 신경성 → (O/C/E/A 중 1개)** 순으로 질문받는다. MBTI가 이미 확신을 준 O·C·E·A는 최대 1문항만 배정되거나, 시간이 부족하면 아예 생략되고 MBTI 사전값이 그대로 사후값으로 유지된다 — 이는 설계상 의도된 동작이다: **정보가 없는 곳에만 시간을 쓴다.**

## 6. 조기종료 규칙

```
STOP WHEN:
  (a) 누적시간 ≥ 5분 (하드 캡), OR
  (b) 전체 가중 불확실성 Ū = (1/8)Σ U_d < θ (θ=0.15 권장), OR
  (c) 최소 5턴(서사형) 충족 AND 모든 U_d < 0.5
```

MBTI 답변과 인터뷰 응답이 일관된 사용자는 조건 (b)/(c)를 빠르게 만족해 **3~4분 내 자연 종료**되고, 모호하거나 MBTI와 실제 언행이 어긋나는 사용자는 5분을 꽉 채운다. "정해진 문항 수 완주"가 아니라 "확신에 도달하면 종료"이므로 지루함 문제가 구조적으로 완화된다.

## 7. 문항뱅크

문항뱅크는 세 계층으로 구성된다. **7.1 공통질문 세트**는 일반적인 생활 맥락에서 차원별 변형을 제공하고, **7.2 연애 맥락 질문 세트**는 같은 차원을 연애 경험·패턴을 소재로 재구성해 데이팅 앱이라는 맥락에 자연스럽게 녹이면서 특히 애착(불안·회피) 차원의 구성타당도를 높인다 — 애착이론 자체가 원래 낭만적 관계를 준거로 삼는 이론이기 때문이다(§17, Brennan, Clark & Shaver, 1998). **7.3 MBTI별 세트**는 §2·§5.4에서 계산된 "이 MBTI 유형이 실제로 헷갈리는 애니어그램 후보 조합"을 정조준한다. 세 세트의 관계와 우선순위 규칙은 §7.4에 정리한다.

### 7.1 공통질문 세트 (전 사용자 공통, 차원별 로테이션)

**회피(avoidance) — 정서적 밀착에 대한 태도**
1. "감정적으로 아주 가까워지는 관계가 되면 오히려 불편해지는 편인가요, 더 편안해지는 편인가요?"
2. "힘든 일이 있을 때, 누군가에게 털어놓는 편인가요, 혼자 정리하고 넘어가는 편인가요?"
3. "연인이 하루 일과를 시시콜콜 궁금해하면 어떤 기분이 드세요?"
4. "관계에서 '적당한 거리감'과 '가까운 밀착' 중 더 편안한 쪽은요?"

**불안(anxiety) — 관계 안정성에 대한 민감도**
1. "연인의 연락이 평소보다 뜸해지면 어떤 생각이 가장 먼저 드시나요?"
2. "상대방이 오늘따라 좀 시큰둥해 보이면, 이유를 계속 생각하게 되는 편인가요?"
3. "'이 사람이 날 정말 좋아하는 게 맞나' 하는 의심이 관계에서 자주 드는 편인가요?"
4. "다투고 난 뒤, 화해하기 전까지 마음이 얼마나 불편하신가요?"

**신경성(N) — 부정 정서의 지속성**
1. "예상치 못한 안 좋은 일이 생겼을 때, 그 감정이 얼마나 오래 가는 편인가요?"
2. "스트레스 받는 상황에서 몸이나 마음에 티가 잘 나는 편인가요?"
3. "하루 중 안 좋았던 일 하나가 나머지 시간 전체 기분을 좌우하는 편인가요?"
4. "화가 나거나 속상할 때, 그 감정이 비교적 쉽게 가라앉는 편인가요?"

**개방성(O) — 새로움에 대한 태도**

- 날개 결합확률 `q(i,w)`(§9.2)는 Big5 사후값만으로 완전히 결정론적으로 계산되며, 이 질문에 대한 답변을 반영하는 별도 우도식은 존재하지 않는다. 즉 이 문구는 **인터뷰 중 묻는 질문이 아니라, 인터뷰 종료 후 결과 화면에서 "이 설명이 맞게 느껴지시나요?" 식으로 보여주는 확인용 UX 카피**다. 시간예산(§4)에도 포함하지 않는다.

```
    최근 미사용 항목을 무작위 선택하되, 직전 질문과 같은 맥락(일반/연애)이
    연속되지 않도록 교차 배치           # 대화 리듬의 자연스러움을 위한 휴리스틱
```

세 세트 모두 §8의 동일한 베이지안 갱신식(정규-정규 켤레, 곱셈적 우도)에 우도 관측치로 투입되며, 통계적 처리는 완전히 동일하다 — 세트 간 차이는 순수하게 **질문의 소재(생활 일반 vs 연애 맥락)와 정조준 대상(범용 vs MBTI별 후보)**일 뿐이다.

## 8. 베이지안 사후분포 갱신 (스트리밍)

매 턴 즉시 갱신되며(배치 처리 아님), 메커니즘은 정규-정규 켤레 및 곱셈적 우도를 사용한다:

```
μ_post(d) = [σ_obs²·μ_prior(d) + σ_prior(d)²·x̄] / [σ_obs² + σ_prior(d)²]
```

**σ_obs (관측 노이즈) 확정값:** `σ_obs = 0.35`. 이는 모든 σ_prior 값(0.12~0.22, §5.1)보다 크게 설정한 것으로, "짧은 인터뷰 한두 문항으로는 사전분포(MBTI 기반)를 강하게 뒤집지 못하고, 사전분포가 사후분포에 계속 상당한 영향력을 유지한다"는 §8 설계 의도를 수치로 구현한다. 문항별 신뢰도(confidence, §6 파싱 스키마)가 낮으면 실제 관측 표준오차를 `σ_obs/confidence`로 확대해 사전분포 쪽으로 더 강하게 수축시킬 것.

**애니어그램 코어 우도함수 L 확정 정의:**
```
p_i^(post) ∝ p_i^(prior) · L(응답 | 코어유형 i)
```
`L`은 인터뷰 응답이 현재 사후분포 상위 2개 후보 유형 중 어느 쪽 성향에 더 가까운지를 나타내는 소프트 대비로 정의한다:
```
L(응답 | 후보유형 i) = exp(k · lean_i),   k = 2
```
`lean_i ∈ [−1,1]`은 파싱된 응답(§6 `normalized_value`, `confidence`)과 문항이 겨냥한 판별축(§7.3 MBTI별 판별질문이면 그 문항이 대조하는 두 후보, 범용/연애맥락 오프너면 현재 사후분포 상위 2개 후보)을 바탕으로 산출하는 부호 있는 정합도이며, 사후분포 상위 2개 외 유형에는 `L=1`(사후분포 불변)을 적용한다.

턴 수가 적으므로(`k`가 작으므로) `σ_obs = σ_item/√k`가 상대적으로 크고, **사전분포(MBTI 기반)가 최종 사후분포에 미치는 영향력이 v1의 10분 설계보다 구조적으로 크다.** 이는 설계상 트레이드오프이며, UI에서 "이 성향은 아직 추정 중이에요"와 같은 확신도 표시 기능의 근거로 남겨둔다.

# Part III — 애니어그램 날개 및 MBTI 재추정

## 9. 날개 조건부확률과 결합분포

### 9.1 정의

애니어그램 원의 인접 규칙에 따라, 코어유형 `i`의 날개는 `{i−1, i+1}` (mod 9, 9와 1이 순환 인접)이다. 인터뷰로 갱신된 Big5 사후값 `b`를 사용해 두 인접 앵커까지의 거리로 조건부확률을 계산한다:

```
d_lo = ‖b − E_{i−1}‖²,   d_hi = ‖b − E_{i+1}‖²
P(wing=i−1 | core=i) = exp(−d_lo/2τ_w²) / [exp(−d_lo/2τ_w²)+exp(−d_hi/2τ_w²)]   ... (3)
τ_w = 0.25   (코어 판별보다 더 촘촘한 구분이 필요하므로 τ_c=0.35보다 낮게 설정)
```

### 9.2 결합확률 (18칸)

```
q(i,w) = p_i · P(w|core=i),     Σ_{(i,w)} q(i,w) = 1                ... (4)
```

이 결합확률이 바로 "3w4가 최고 확률이지만 7w8도 무시 못할 확률로 남는" 다봉적 결과를 자연스럽게 만들어낸다 — **코어유형 사후분포 `p` 자체가 이미 다봉적**이라면(예: 3유형과 7유형이 비슷하게 유력), 각 코어 안에서의 날개 조건부확률이 곱해져 서로 다른 코어에서 나온 두 조합이 순위표 상단에 나란히 놓이게 된다. 별도의 "복수유형 허용" 로직을 추가할 필요 없이 §9.1~9.2의 곱셈 구조만으로 충분하다.

**실계산 예시** (인터뷰 후 Big5 사후값이 3유형·7유형 원형 사이에 위치한 가상 사례):

```
코어유형 사후분포: 7유형 18.0%, 3유형 17.8%, 8유형 16.6%, ...
→ 결합확률 상위: 8w7 14.4%,  7w8 14.3%,  3w2 13.0%,  2w3 8.9%, ...
```

8w7과 7w8이 사실상 동률로 상위권에 공존한다 — 서로 다른 코어유형(8과 7)에서 나온 조합이 근소한 차이로 나란히 남는 구조가 그대로 재현된다.

### 9.3 날개 번호 주변확률

궁합 계산(§11.2)을 위해 "날개로서 어떤 번호가 얼마나 등장하는지"를 코어와 무관하게 합산한다:

```
r(k) = Σ_{i : k∈wings(i)} q(i,k),    k=1,...,9,   Σ_k r(k)=1          ... (5)
```

## 10. MBTI 축 재추정 (보너스)

인터뷰로 갱신된 Big5 사후값을 역방향으로 매핑해 "AI가 대화에서 감지한 성향"을 산출한다:

```
P(축=E) = σ_logistic( (E_post − 0.5) / s ),    s = 0.15               ... (6)
```

(N/S↔O, T/F↔A, J/P↔C에 동일 구조 적용.) 자기보고 MBTI와의 불일치가 클 경우("입력하신 MBTI는 ENFP, 대화에서 감지된 성향은 ENFJ 78%") UI 상 흥미 요소로 노출할 수 있다. **이 값은 궁합 점수 계산에는 관여하지 않는 순수 출력 기능**이다.

## 11. 네 부분점수

### 11.1 Big Five, 스턴버그, 애착 (v2와 동일 — 전개식 전체 복원)

**Big Five — 유사도항 + 절대수준항:**
```

S_B5,sim = (cos(c_A,c_B)+1)/2
Readiness_X = 0.4·(1 − N_X) + 0.3·A_X + 0.3·C_X                     ... (11a)
S_B5 = 0.4·S_B5,sim + 0.6·[(Readiness_A+Readiness_B)/2]
```

**스턴버그 삼각형 — 성향 추정치:**
```
I_X = 0.5·A_X + 0.3·p_X(하트그룹 2,3,4 합) + 0.2·(1 − N_X)
P_X = 0.5·E_X + 0.5·O_X
C_X = 0.6·C_X(성실성) + 0.4·(1 − avo_X)                              ... (11b)
t^trait_X = (I_X, P_X, C_X)

t_X = 0.6·t^state_X + 0.4·t^trait_X
S_ST = 1 − ‖t_A−t_B‖/√3
```

**애착 — 정식 가중항:**
```
Stability_X = 1 − 0.5·(anx_X + avo_X)
PursueWithdraw = anx_A·avo_B + anx_B·avo_A                           ... (11c)
S_ATT = clamp(0.5·(Stability_A+Stability_B) − 0.5·PursueWithdraw, 0, 1)
```

## 12. 원점수 결합과 Monte Carlo 재캘리브레이션

가중치는 v2와 동일하게 유지한다 (변경된 것은 `S_EN` 내부 구성뿐이므로 상위 가중치 재조정은 불필요):

```
S_raw = 0.25·S_B5 + 0.15·S_EN + 0.30·S_ST + 0.30·S_ATT                  ... (9)
```

`S_EN`의 정의가 바뀌었으므로 `μ̂, σ̂`는 10만 쌍 재시뮬레이션(날개 주변확률 계산 포함)으로 다시 추정했다:

```
μ̂ = 0.5259   (v2: 0.5316 — 날개 보너스 항이 평균을 소폭 낮춤)
σ̂ = 0.0726   (v2: 0.0727 — 거의 변화 없음)
k  = 25/2.5758 ≈ 9.7056

DNA% = clamp( 75 + 9.7056·(S_raw − 0.5259)/0.0726 , 50, 100 )           ... (10)
```

검증: 클리핑 이전 원점수가 50 미만 0.127%, 100 초과 0.369% (합계 0.496%, v2와 거의 동일한 근사 정규성 유지).

## 13. 전체 알고리즘

```
FUNCTION analyze_person_v21(mbti, interview_session):
    b0 ← big5_prior(mbti);  p0 ← enneagram_core_prior(b0)
    ψ0 ← (b0.N, 1-(b0.E+b0.A)/2)
    U ← compute_uncertainty(b0, p0, ψ0)          # §5

    WHILE elapsed < 5min AND NOT early_stop(U):   # §6
        target ← argmax(U, unasked_dims)
        response ← ask(target)                    # §7 문항뱅크
        (b0,p0,ψ0) ← bayesian_update(target, response)  # §8
        U ← recompute_uncertainty(...)
        ask_fixed_sternberg_items_if_not_yet_asked()     # §4 유형 C, 엔트로피 무관 고정 포함

    q ← enneagram_wing_joint(b0, p0)              # §9
    r ← wing_marginal(q)
    mbti_est ← invert_to_mbti(b0)                 # §10 (출력 전용)

    RETURN { b: b0, p: p0, ψ: ψ0, q, r, t_state, mbti_est }
END FUNCTION

FUNCTION compute_romantic_dna_v21(person_A, person_B):
    profile_A ← analyze_person_v21(A.mbti, A.interview)
    profile_B ← analyze_person_v21(B.mbti, B.interview)

    S_B5  ← big5_score(profile_A, profile_B)                # §11.1
    S_EN  ← (1-0.08)·(p_A^T M p_B) + 0.08·wing_bonus(r_A,r_B) # §11.2
    S_ST  ← sternberg_score(profile_A, profile_B)            # §11.1
    S_ATT ← attachment_score(profile_A, profile_B)           # §11.1

    S_raw ← 0.25·S_B5 + 0.15·S_EN + 0.30·S_ST + 0.30·S_ATT
    z ← (S_raw − 0.5259) / 0.0726
    DNA_pct ← clamp(75 + 9.7056·z, 50, 100)

    RETURN DNA_pct, { S_B5, S_EN, S_ST, S_ATT, profile_A.mbti_est, profile_B.mbti_est }
END FUNCTION
```

## 16. 한계

1. **연속-이산 엔트로피 비교의 근사성.** §5에서 정규분포의 분산과 심플렉스의 섀넌 엔트로피를 각각 정규화해 같은 척도에서 비교하는 것은 엄밀한 정보이론적 등가가 아니라 실용적 근사다. 완전한 해법은 두 경우 모두를 기대정보이득(expected information gain) 프레임으로 통일하는 것이나, 실시간 5분 인터뷰에 요구되는 계산 비용 대비 이득이 크지 않아 근사를 채택했다.
2. **σ_prior 차등화 값(0.12~0.22)은 문헌의 상관계수 크기 순서를 반영한 근사치**이며, 정밀한 베이지안 사전분포 폭 추정이 아니다. 실사용자 데이터가 쌓이면 각 차원의 실제 예측오차 분산으로 재보정해야 한다.
3. **날개 궁합 가중치 δ=0.08은 임의값**이다. §14 예제처럼 실제 영향은 미미한 수준으로 설계되었으나, 사용자 반응(예: "날개까지 반영된 궁합 설명"에 대한 만족도)에 따라 조정 여지가 있다.
4. **나머지 한계(MBTI 신뢰도, Big5 유사성의 약한 예측력 등)는 v1 §10, v2 §17과 동일하게 적용된다.**
