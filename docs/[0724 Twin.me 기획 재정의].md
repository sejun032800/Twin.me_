[Twin.me 대규모 개편 (0724)]

1. 목적
그동안 나름 잘 진행되던 Twin.me를 대규모 개편하는 이유는,
    (1) 0722 투자 미팅 시 받은 피드백에 대한 수정
    (2) 서비스의 방향성 자체를 다시 확립
    (3) 불필요한 기능과 부가적인 시스템을 제거하여 MVP 앱의 목적에 부합하게 (앱을 가볍게 유지)
    (4) 서비스 톤 수정 : 힙하고 팬시한 느낌으로 변경 (현재는 진지하고 고급스러운 느낌)
    하기 위함이다.

2. 앱의 방향성
기존 Twin.me는 '나를 돌아보는 거울이자 아카이브'였다.
하지만 투자 미팅을 진행해보니, 비용 문제와 실질적인 효과에 대한 의문이 제기되었고, 이는 기획자 본인 역시 수정해야할 필요성을 느꼈다.
따라서 Twin.me 앱의 방향성을 수정한다.

방향성 : 커플끼리 '재미'로 할 수 있는 가벼운 연인 관계 측정 앱
    (1) 사람들이 마치 신년 운세, 사주에 돈을 지불하는 것처럼, 혹은 퍼스널 컬러 진단같이 스스로를 알 수 있는 자가성찰에 대한 지불 의사가 높음.
    (2) '우리 사이는 연인 상위 몇 프로일까?'하며 관계를 게이미피케이션하여 호기심 자극
    (3) 채팅을 통해 일일, 주간 결산 리포트를 발급. 연인 간 흥미 유발

즉, 사용자가 이 앱을 찾는 경우는
    (1) 자신의 연애 성향을 파악하고 싶거나
    (2) 연인과의 대화를 통해 리포트를 받아보고 싶거나
    (3) 우리 커플의 'DNA 일치율'을 알고 싶거나
이다.

4. 주요 구성

- 온보딩 : 
    (1) 로고, 밤하늘 감성 페이지 : 로고가 뜨며 자연스럽게 별이 빛나는 밤하늘로 이동, 유저와 연인의 세계라는 설정. 연인 간의 공유된 소중한 추억은 밤하늘이 되고, 다정했던 말들은 별똥별이 되어 마음 속에 흐른다.
        [1] 시작하기 : 회원가입, 로그인, 소셜로 로그인, (개발자 페이지)


    (2) 기본정보 입력 (이름, 성별, 생년월일, MBTI) : MBTI를 모를 경우에 (외향적/내향적, 감각형/직관형, 감정형/사고형, 즉흥형/계획형의 여부를 물어봄 (이게 맞는지 모르겠지만, 이런 느낌으로))

    (3) 유저 5분 통화(스킵 가능, but 정확도 떨어짐 고지) : MBTI를 토대로, 애니어그램과 빅5를 추론 가능하도록 하는 로직을 발동시킴 (이미 구현한 베이지안 추론 엔진 발전시키기), 사랑의 삼각 이론 등도 추출함.

    (4) 결과 랜딩 페이지 : 화려하고 감성적인 UI로 스스로에 대한 정보가 나옴. 사용자의 성격 유형 분석 : MBTI와 질문을 통해 추론한 빅5 유형과 애니어그램 유형 <가장 높은 확률을 제시, 후보 2개정도 (-- 일수도 있어요!)제시, 그리고 '결과와 달라요' 버튼 만들어서 사용자가 수정 가능하게끔>을 분석해서, 아래 결과를 보여줌.
        [1] 사용자의 성격 유형 : mbti(만약 역추론 결과가 다르다면 'AI 재추정 MBTI 리빌'), 빅5(막대그래프로), 애니어그램, 
        [2] 사용자의 연애 유형 : 연애 유형 (*이름만 갖다 붙이기(애니어그램 코어 x 애착 패턴)) 제공
        [3] 사용자의 연애 유형 자료 : 사용자의 연애 유형의 대표적 인물, 해당 유형의 평균적인 연애 일치율, 해당 유형과의 best, worst 연애 유형들

    (5) 연인 초대 화면 : 연인을 초대하면 더욱 자세한 리포트를 받을 수 있다고 하여 연인 초대 유도. 현행대로 초대코드 발급-입력 시 1대1 매칭 방식 채택. (나중에 할게요, 솔로예요 버튼 만들기) 이후 감성적인 연출과 함께 메인 4개 탭으로 진입.


[메인 탭 : 지구 표면과 별똥별들]
- 전제 : 아래의 4개의 탭들은 기존 4개의 수평 탭으로 병렬 구조를 가지고 있었다. 이 부분을 변경한다.
    (1) 메인 컨셉(지구) : 온보딩 이후 '나'라는 평행 지구에 도착했다. 각 4개의 탭은 4분할로 이루어진 원형의 지구다. 즉, 수평 이동을 할 때마다 지구가 '적당히' 돌아가며, 지구를 스와이프하며 탭을 이동한다.
    (2) 나머지 하늘 공간 : 기본적으로 6축 아우라 테마가 작동하여 사용자마다 다른 하늘의 색이 펼쳐진다. 별똥별이 간간히 떨어지며, 별똥별의 꼬리에는 연인과 나눴던 다정한 말들이 이어진다. 탭하면 별을 수집할 수 있다. 
        [1] 별똥별은 다정한 대화를 나누면 나눌 수록 많이(정비례는 아님) 떨어지며, 말의 분위기에 따라 별의 색이 다르다. (네온 계열)
        [2] 별을 수집할 수 있으며, 수집한 별은 밤하늘 탭에서 볼 수 있다. 

- 홈 탭(주거지역의 지구) : 앱의 홈 화면.
    (1) 연애 DNA 일치율 : 이 앱의 핵심 축. 일치율을 누르면 화려한 그래픽으로 카드 형태의 DNA 일치율과 티어 등 정보가 나온다. (티어 그래픽 + 점수 변화 스토리 문장)
    (2) 헤더 : 연인 (이름+하트+이름), 연애 D-DAY
    (3) 미연동 시 초대 배너: "연인을 초대해보세요" → 탭하면 설정 탭으로 이동.


- 채팅 탭(데이트 거리의 지구) : 채팅 탭이다.
    (1) 연인 탭 : 연인과 실시간 채팅을 나눌 수 있도록 하는 채팅 탭.
    (2) 연인 탭 링(스토리) : 인스타그램 스토리처럼 연인사이에 스토리를 보낼 수 있는 기능.
    (3) 트윈 방 : 평행
